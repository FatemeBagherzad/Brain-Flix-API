# BrainFlix API !

This repository contains the backend API for BrainFlix App, which powers the application's core functionality.

## Technologies

<li>Node.js</li>
<li>Express</li>
<li>REST APIs</li>
<li>Multer</li>
<li>CORS</li>
<li>Nodemon</li>
<li>Git, GitHub</li>
